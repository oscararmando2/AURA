# Video Assets

## Required Video File

Please download the following video file and place it in this directory:

**File Name:** `pilates-background.mp4`

**Download URL:** https://www.pexels.com/video/8746842/download/

**Video Details:**
- Video ID: 8746842
- Source: Pexels
- Resolution: 1920x1080 (HD) recommended
- The video will automatically play from second 1 to second 25 and loop

**Download Instructions:**

### Option 1: Direct Download (Recommended)
1. Visit https://www.pexels.com/video/8746842/download/
2. The video will download automatically
3. Rename the downloaded file to `pilates-background.mp4`
4. Move/copy it to this `assets/videos/` directory

### Option 2: Via Pexels Website
1. Visit https://www.pexels.com/video/8746842/
2. Click the "Download" button (green button on the page)
3. Select HD quality (1920x1080)
4. Rename the downloaded file to `pilates-background.mp4`
5. Move/copy it to this `assets/videos/` directory

### Option 3: Using curl or wget (Command Line)
```bash
# Using curl
curl -L -o pilates-background.mp4 "https://www.pexels.com/video/8746842/download/"

# Using wget
wget -O pilates-background.mp4 "https://www.pexels.com/video/8746842/download/"
```

**Usage:**
The video will be used as a fullscreen background for the hero section (welcome section) on the homepage. The text "Bienvenida", "Pilates a tu medida", and "Fortalece tu cuerpo y mente con Pilates" will appear in the foreground over the video.

**Note:** If the video file is not present, the page will display the gradient background as a fallback.
